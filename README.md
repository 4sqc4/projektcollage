# 🏦 Проект: Консольный Банк (Bank Management System)

Учебный проект по изучению Объектно-Ориентированного Программирования (ООП) на языке Python. Проект имитирует работу банковской системы с пользователями, счетами и базовой системой безопасности.

---

## Особенности реализации
* **Модульная структура:** Каждый класс вынесен в отдельный файл согласно требованиям.
* **Применение ООП:** Использование абстрактных классов, наследования, инкапсуляции и полиморфизма.
* **Кастомные исключения:** Обработка ошибок через собственные классы исключений.
* **Система безопасности:** Ограничение попыток входа, ночной "тихий" режим и мониторинг крупных сумм.

---

## Структура проекта
* `abstract_account.py` — Абстрактный класс для банковских счетов.
* `bank_account.py` — Реализация базового счета.
* `client.py` — Класс клиента (пользователя).
* `bank.py` — Главный управляющий класс банка (база данных, безопасность).
* `exceptions.py` — Пользовательские ошибки (`AccountFrozenError` и др.).
* `main.py` — Интерактивное меню управления через консоль.
